Log4j.
Для вывода инфы в файл, используя LOG4J:
1. Подключаем dependency
2. Создаем файл в ресурсах log4j.properties с указанием деталей, прим., уровень логирования, куда сохранять, объем и т.п.
3. Для логирования задействуем реализацию с использованием jdbc, прим. UserDaoImpl
В необходимых "местах" прописываем логирование
4. В основном Spring-классе создаем объект логгера, при выводе инфы задействуем его
